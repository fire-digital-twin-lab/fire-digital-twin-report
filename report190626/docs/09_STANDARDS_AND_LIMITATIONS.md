# 09 — Standards and Limitations / Quy chuẩn, tiêu chuẩn và giới hạn

## VI — Vai trò của quy chuẩn/tiêu chuẩn

Quy chuẩn/tiêu chuẩn dùng để:

- tham chiếu ngưỡng tenability/hazard;
- tham chiếu hệ báo cháy;
- tham chiếu sprinkler;
- tham chiếu hút khói/tăng áp;
- tránh tự đặt ngưỡng tùy tiện;
- giúp báo cáo có cơ sở kỹ thuật.

Không dùng tiêu chuẩn để tuyên bố prototype đạt thẩm duyệt PCCC.

## EN — Role of standards

Standards are used to:

- reference tenability/hazard thresholds;
- reference fire alarm systems;
- reference sprinkler systems;
- reference smoke exhaust/stair pressurization;
- avoid arbitrary thresholds;
- provide technical grounding.

Do not use standards to claim that the prototype satisfies fire-safety approval.

## VI — Ghi chú bắt buộc về TCVN báo cháy

Trước khi nộp báo cáo, phải kiểm tra bản hiện hành của tiêu chuẩn báo cháy. Nếu TCVN 5738:2021 đã được thay thế, ưu tiên cập nhật theo TCVN 7568-14:2025 hoặc bản mới nhất được cơ quan có thẩm quyền công bố.

Không trích số ngưỡng nếu chưa xem văn bản gốc.

## EN — Mandatory note about Vietnamese fire alarm standards

Before submission, verify the current Vietnamese fire alarm standard. If TCVN 5738:2021 has been replaced, update the report using TCVN 7568-14:2025 or the latest official version.

Do not quote numerical thresholds without checking the original document.

## VI — Giới hạn nghiên cứu

| Limitation | Meaning |
|---|---|
| No real fire data | evaluation is against simulation outputs |
| No real IoT hardware | IoT is simulated |
| CFAST is zone-level | not 3D field prediction |
| FDS is selected validation | not bulk dataset source |
| Hazard labels may be proxy | unless standards-backed |
| AI is surrogate | not CFD replacement |
| Dashboard is research demo | not real control system |
| No legal conclusion | no regulatory approval claim |
